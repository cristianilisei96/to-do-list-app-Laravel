# ✅ Laravel To‑Do List App

A clean and simple To‑Do List application built with **Laravel 10**, **Bootstrap 5**, and **MySQL**.  
Users can register, log in, create tasks, edit them, mark them as completed/uncompleted, view details, and delete them.  
The UI is responsive, minimalistic, and optimized for a smooth user experience.

---

## 🚀 Features

- User authentication (Register / Login / Logout)
- Add new To‑Dos (title + description)
- Mark tasks as **Completed** or **Uncompleted**
- Edit tasks **only when they are Uncompleted**
- Delete tasks
- View task details
- Clean Bootstrap 5 interface
- Success alerts for all actions
- Line‑through styling for completed tasks
- Fully responsive layout

---

## 🛠️ Technologies Used

- **Laravel 10**
- **PHP 8+**
- **MySQL**
- **Bootstrap 5**
- **Vite**
- **Blade Templates**

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/to-do-list-app-Laravel.git
cd to-do-list-app-Laravel

Install PHP dependencies:
composer install

Install Node dependencies:
npm install

Copy .env:
cp .env.example .env

Generate app key:
php artisan key:generate

Configure database in .env, then run:
php artisan migrate

Start the dev servers:
php artisan serve
npm run dev
```
